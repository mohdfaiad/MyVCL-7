In Delphi/CB, select File -> Open, browse in \Install\*\, and, one by one, open all the project group files respecting the order.
If you don't see the package files, click View -> Project Manager.
Firstly Build all the RunTime packages (the ones which start with "dx" or "cx").
Then Install all the DesignTime packages (the ones wich start with "dcl").
From Delphi/CB 2007 you can select and Build/Install more than one package at a time.
