In this demo you can:

 - Create/delete/edit events (use popupmenus, inplace editor and dialogs)

 - Generate hundreds of random events (use the button at the bottom right corner of the form)

 - See and use different integrated popup menus for different scheduler areas

 - Choose different views (View menu: Day/Work week/Week/Month/Timegrid/Year. As an alternative to the Day/Work week/Week/Month menu items select days in the Date Navigator)

 - Change the scheduler's layout (View menu: Date Navigator/Control Box/View position; you can also use the splitters located between the View, Date Navigator and Control Box)

 - See how the scheduler changes its appearance depending on the Look&Feel used (Look&Feel menu)

- Menu File/Export:
	- export data in either Excel's format, as text, html or xml

- Menu File/Synchronization:
	- synchronize the scheduler with MS Outlook®
	- synchronize MS Outlook® with the scheduler

- Menu Options/Day:
	- hide the all-day event area
	- make event times permanently visibile regardless of whether they match the timescale's units or not
	- hide the non-working hours
	- show minutes in a time ruler
- Menu Options/Week:
	- compress weekends
	- switch between either one or two columns of week view days
- Menu Options/Month:
	- compress weekends
- Menu Options/Timegrid:
	- hide the non-working hours
- Menu Options/Year:	
	- show only all-day events

- Menu Events/Intersection:
	- schedule overlapped events
- Menu Events/Sharing between resources:
	- allow events to be assigned to specific resources

- Menu Resources/GroupBy:
	- show the time view without resources
	- view events organized first by resources and then by dates
	- view events organized first by dates and then by resources 
- Menu Resources/LayoutEditor...:
	- rearrange resources using the Resources layout editor
- Menu Resources/Resources per page:
	- show all resources in the time view
	- group either one, two or three resources per page


The following features have been added to ExpressScheduler 2:

- Timegrid and year views

- Reminder 

- Resource scrolling and grouping resources into pages

- The ability to schedule events sequentially or simultaneously

- Sharing events between specific resources

- Cloning and moving events using the scheduler clipboard

- Event custom icons

- Event custom labels

- Event hint customization

- The ability to modify the look and feel of all the scheduler's dialogs

- Resource header icons

- Multiline resource captions

- Day view:
	- The ability to hide the non-working hours in the time view
	- The ability to hide the all-day event area
	- The ability to optionally show minutes in a time ruler
	- The ability to permanently show event times regardless of whether they match the timescale's units or not

- Week view:
	- The ability to arrange days into either one or two columns

- The ability to plug a standalone Date Navigator into the scheduler. Once connected, it acts as the scheduler's built-in Date Navigator

- A flexible range of date selection in a standalone Date Navigator. End-users can select either up to fourteen days (or six weeks) or an unlimited number of days

- Bidirectional synchronization between the scheduler and MS Outlook®

- Smart refresh mode - if it's switched on, the scheduler only loads the updated values rather than all the data

- The ability to filter the events that are loaded into the view from storage

- When in smart refresh mode the ability to manually load values updated by another control in the shared store 

- Improved query performance

- The scheduler's report:
	- Yearly print style
	- Support for the new features which have been added to the time views
